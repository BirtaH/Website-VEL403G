Design

The group decided to make a ice cube mould, with icecubes shaped lika a snowflake. The ice cube mould should be able to create 15 cubes. This idea was inspired by a picture found online. This design was inspired by a picture found online. The picture can be seen here.

The video How to Create a Two-Part Mold in Fusion 360 - Learn Autodesk Fusion 360 in 30 Days: Day #21 explains in detail how to create a mould for a object in Fusion. The first step is to create the object and then the mould is created. My contribution to the project was the design and drawing of the ice Cube tray. Before I started drawing I watched the video How to 3D Model an Ice Cube Tray - Learn Autodesk Fusion 360 in 30 Days: Day #5

I started by drawing a box that the mould will be created from. I choose the size 300x200mm. Then fillet is used to round the corners.

Next I drew a single snowflake. This was done using line and mirror. I started by drawing one quarter of the snowflake and then mirroring it using a construction line.

Then the using extrude the shape is removed from the box. To make removal of icecubes from the mould easier the cut has a draft angle of 1°. The settings of the extrude can be seen on the picture here below.

Fillet is used to round the edges.

Using rectangle pattern the snowflakes can be replicated 15 times (3x5 snowflakes).

I had some trouble using shell with so many snowflakes so I had to remove some from the mould. My computer would crash every time I tried to use shell. In the end it was 6 snowflakes(2x3) and then the command shell is used. I chose wall thickness 3mm and the result can be seen here below. .

Using fillet the edges of the mould are rounded and the final prodect can be seen here below.
